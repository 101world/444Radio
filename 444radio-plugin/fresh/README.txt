=============================================
  STANDALONE MODE (No DAW needed!)
=============================================
1. Double-click "444 Radio Standalone.exe"
2. The app opens a window with the full 444 Radio studio
3. Log in with your 444 Radio token (from Settings on the website)
4. Requires WebView2 Runtime (most Windows 10/11 PCs already have it)
   Download if needed: https://developer.microsoft.com/en-us/microsoft-edge/webview2/

NOTE: The standalone .exe does NOT need installation — just run it.
      You can put it anywhere (Desktop, a USB drive, etc.)

=============================================
  VST3 PLUGIN (For DAWs)
=============================================
1. Copy the entire "444 Radio.vst3" folder to:
   C:\Program Files\Common Files\VST3\
2. Restart your DAW
3. Scan for new plugins if needed

SUPPORTED DAWs:
- Ableton Live 11+
- FL Studio 21+
- Cubase 12+
- Bitwig Studio 5+
- Any VST3-compatible DAW

=============================================
  REQUIREMENTS
=============================================
- Windows 10/11 (64-bit)
- WebView2 Runtime (usually pre-installed)
- Internet connection (for AI generation)
- 444 Radio account with credits

=============================================
  GETTING YOUR PLUGIN TOKEN
=============================================
1. Go to https://444radio.co.in and sign in
2. Go to Settings (gear icon)
3. Scroll to the "Plugin" section
4. Copy your token (starts with "444r_")
5. Paste it in the plugin's token field
